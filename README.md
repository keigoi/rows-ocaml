## Installation and development

Trying with utop:

```sh
git clone http://github.com/keigoi/rows-ocaml.git
cd rows-ocaml
dune utop
```

Installation:

```sh
opam install http://github.com/keigoi/rows-ocaml.git
```

Uninstall:

```sh
opam remove rows
```
